## What is Text Shot?

Text Shot is a lightweight macOS screenshot-to-text utility that lets you capture any screen region and instantly extract text using on-device OCR. It runs entirely on-device and is built for speed, simplicity, and keyboard-driven workflows.

## How It Works

The workflow is simple:

1. **Trigger**: Press your keyboard shortcut (default: `⇧⌘2`)
2. **Select**: Click and drag to select the screen region containing text
3. **Extract**: Text Shot performs OCR on the selected region
4. **Paste**: The extracted text is automatically copied to your clipboard

The entire process takes just seconds, with no manual file saving or uploading required.

## Technical Architecture

Text Shot is built as a native Swift menu bar application with the following components:

```swift
// Core capture flow from AppController.swift
func runCaptureFlow() async {
    // 1. Check Screen Recording permission
    guard hasScreenCaptureAccess else { return }
    
    // 2. Capture selected region using screencapture
    let capture = await captureService.captureRegion()
    guard let imagePath = capture.path else { return }
    
    // 3. Run OCR with automatic fallback
    let text = try ocrService.runOcrWithRetry(imagePath: imagePath)
    
    // 4. Copy to clipboard
    clipboardService.write(text)
    
    // 5. Show confirmation (if enabled)
    showToastIfEnabled("Copied!")
}
```

**Key components:**

- **CaptureService** - Uses macOS `screencapture` utility for region selection
- **OCRService** - Apple Vision framework with automatic quality fallback
- **HotkeyManager** - Global hotkey registration using KeyboardShortcuts library
- **ClipboardService** - NSPasteboard integration for clipboard access
- **ToastPresenter** - Visual confirmation overlay with native blur effects

### OCR Strategy

Text Shot employs a multi-level OCR strategy to maximize text recognition accuracy:

```swift
// From OCRService.swift
private let retryChain: [(RecognitionLevel, Bool)] = [
    (.accurate, true),   // Try accurate mode with language correction
    (.accurate, false),  // Try accurate mode without correction
    (.fast, true)        // Fall back to fast mode if needed
]
```

This ensures high-quality results even with challenging input like low-resolution screenshots or stylized fonts.

## Privacy & Security

<Note>
  Text Shot requires Screen Recording permission to capture selected regions. This is a standard macOS permission required by all screenshot utilities.
</Note>

Text Shot is designed with privacy as a core principle:

- **No network access** - The app never makes network requests during normal operation
- **No data collection** - Your captures and extracted text stay on your device
- **No analytics** - Zero telemetry, tracking, or usage statistics
- **No accounts** - Install and use immediately without creating an account
- **Local processing only** - All OCR happens on-device using Apple's Vision framework

The only permission required is **Screen Recording**, which is necessary to capture the screen region you select. macOS will prompt you to grant this permission on first launch.
