## Welcome to Text Shot

Text Shot is a lightweight macOS menu bar application that transforms screenshots into editable text instantly. Capture any screen region with a keyboard shortcut, and Text Shot extracts the text using Apple's on-device Vision framework—no internet connection required.

## Key Features

## How It Works

<Steps>
  <Step title="Press your hotkey">
    Use your configured keyboard shortcut (default: Cmd + Shift + 2) to activate capture mode
  </Step>
  <Step title="Select the region">
    Drag to select any area of your screen containing text
  </Step>
  <Step title="Text extracted automatically">
    Text Shot uses Apple Vision to recognize text and copies it to your clipboard
  </Step>
  <Step title="Paste anywhere">
    The extracted text is ready to paste into any application
  </Step>
</Steps>

## System Requirements

<Note>
Text Shot requires macOS 13 (Ventura) or later and Screen Recording permission to capture screen regions.
</Note>
